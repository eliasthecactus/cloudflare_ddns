> 😋 A simple Bash script to update cloudflares dns entry with your current IP

## Install
```sh
git clone https://github.com/eliasthecactus/cloudfalre_ddns.git
cd cloudfalre_ddns/
sudo chmod +x updater.sh
```

## Usage
```sh
./updater.sh
```
